## Usage

Import the component:

```tsx
import { Spinner } from "@fan-ui/solid";
```

Basic spinner:

```tsx
<Spinner />
```

With size:

```tsx
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
<Spinner size="xl" />
```

With text:

```tsx
<div class="flex items-center gap-2">
  <Spinner size="sm" />
  <span class="text-sm text-muted-foreground">Loading...</span>
</div>
```
