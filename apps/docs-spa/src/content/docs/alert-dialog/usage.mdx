## Usage

Import the components:

```tsx
import {
  AlertDialog,
  AlertDialogTrigger,
  AlertDialogContent,
  AlertDialogHeader,
  AlertDialogFooter,
  AlertDialogTitle,
  AlertDialogDescription,
  AlertDialogCancel,
  AlertDialogAction,
} from "@fan-ui/solid";
```

Basic usage:

```tsx
<AlertDialog>
  <AlertDialogTrigger>Open</AlertDialogTrigger>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Are you sure?</AlertDialogTitle>
      <AlertDialogDescription>This action cannot be undone.</AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel variant="outline">Cancel</AlertDialogCancel>
      <AlertDialogAction variant="destructive">Delete</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```

### Root Provider

Use <InlineCode>AlertDialogBase.RootProvider</InlineCode> when you need to control the dialog state from outside its component tree. This pattern uses the <InlineCode>useDialog</InlineCode> hook from Ark UI and is useful for utility dialogs like delete confirmations that can be triggered from any element.

<AlertDialogDeleteDemo />

```tsx
import { useDialog, type UseDialogReturn } from "@ark-ui/solid/dialog";
import { AlertDialogContent, AlertDialogHeader, AlertDialogFooter, AlertDialogTitle, AlertDialogDescription, AlertDialogCancel, AlertDialogAction, AlertDialogBase } from "~/components/alert-dialog";
import { Button } from "~/components/button";
import { type Component } from "solid-js";

interface DeleteAlertDialogProps {
  dialog: UseDialogReturn;
  title: string;
  description: string;
  onDelete: () => void;
}

const DeleteAlertDialog: Component<DeleteAlertDialogProps> = (props) => {
  return (
    <AlertDialogBase.RootProvider value={props.dialog}>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>{props.title}</AlertDialogTitle>
          <AlertDialogDescription>{props.description}</AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogCancel variant="outline">Cancel</AlertDialogCancel>
          <AlertDialogAction variant="destructive" onClick={() => props.onDelete()}>
            Delete
          </AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialogBase.RootProvider>
  );
};

export function DeleteConfirmation() {
  const dialog = useDialog({ defaultOpen: false });

  return (
    <div>
      <Button onClick={() => dialog().setOpen(true)} variant="destructive">
        Delete Account
      </Button>
      <DeleteAlertDialog
        dialog={dialog}
        title="Delete Account"
        description="This will permanently delete your account and remove your data from our servers."
        onDelete={() => dialog().setOpen(false)}
      />
    </div>
  );
}
```

The key difference:

- **<InlineCode>AlertDialog</InlineCode>** — manages its own state internally. Use for simple, self-contained usage.
- **<InlineCode>AlertDialogBase.RootProvider</InlineCode>** — accepts a pre-created context via <InlineCode>useDialog</InlineCode>. Use when you need to read or control the dialog state from outside the component tree. Ideal for reusable utility dialogs.

### Anatomy

| Part | Element | Description |
|------|---------|-------------|
| <InlineCode>AlertDialog</InlineCode> | — | Manages dialog state, uses <InlineCode>role="alertdialog"</InlineCode> |
| <InlineCode>AlertDialogTrigger</InlineCode> | <InlineCode>button</InlineCode> | The button that opens the dialog |
| <InlineCode>AlertDialogContent</InlineCode> | <InlineCode>div</InlineCode> | The dialog panel (portal to body) |
| <InlineCode>AlertDialogHeader</InlineCode> | <InlineCode>div</InlineCode> | Header section for title and description |
| <InlineCode>AlertDialogTitle</InlineCode> | <InlineCode>h2</InlineCode> | The dialog title |
| <InlineCode>AlertDialogDescription</InlineCode> | <InlineCode>div</InlineCode> | Descriptive text for the dialog |
| <InlineCode>AlertDialogFooter</InlineCode> | <InlineCode>div</InlineCode> | Footer section for action buttons |
| <InlineCode>AlertDialogCancel</InlineCode> | <InlineCode>button</InlineCode> | Dismisses the dialog |
| <InlineCode>AlertDialogAction</InlineCode> | <InlineCode>button</InlineCode> | Confirms the dialog action |
