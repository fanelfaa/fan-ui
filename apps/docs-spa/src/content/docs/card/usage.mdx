import { InlineCode } from "@fan-ui/solid";

## Usage

Import the components:

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from "@fan-ui/solid";
```

Basic card:

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Content</p>
  </CardContent>
  <CardFooter>
    <p>Footer</p>
  </CardFooter>
</Card>
```

## Anatomy

A card is composed of the following parts:

| Part | Element | Description |
|------|---------|-------------|
| <InlineCode>Card</InlineCode> | <InlineCode>div</InlineCode> | The root container. |
| <InlineCode>CardHeader</InlineCode> | <InlineCode>div</InlineCode> | The header section, typically contains the title and description. |
| <InlineCode>CardTitle</InlineCode> | <InlineCode>h3</InlineCode> | The title text. |
| <InlineCode>CardDescription</InlineCode> | <InlineCode>p</InlineCode> | The description text, styled as muted foreground. |
| <InlineCode>CardContent</InlineCode> | <InlineCode>div</InlineCode> | The main content area. |
| <InlineCode>CardFooter</InlineCode> | <InlineCode>div</InlineCode> | The footer section, typically used for actions. |
