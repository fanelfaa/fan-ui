## With Indicator

<InlineCode>ToggleIndicator</InlineCode> renders different content based on the toggle state — the <InlineCode>fallback</InlineCode> is shown when off, and the children are shown when on.

<ToggleIndicatorDemo />

```tsx
import { Toggle, ToggleIndicator } from "~/components/toggle";

export function ToggleIndicatorDemo() {
  return (
    <div class="flex items-center gap-4">
      <Toggle>
        <ToggleIndicator fallback={<SunIcon />}>
          <MoonIcon />
        </ToggleIndicator>
      </Toggle>
      <Toggle defaultPressed>
        <ToggleIndicator fallback={<SunIcon />}>
          <MoonIcon />
        </ToggleIndicator>
      </Toggle>
    </div>
  );
}
```

## Usage

Import the components:

```tsx
import { Toggle, ToggleIndicator } from "~/components/toggle";
```

Basic:

```tsx
<Toggle>
  <BoldIcon />
</Toggle>
```

Default pressed:

```tsx
<Toggle defaultPressed>
  <BoldIcon />
</Toggle>
```

Controlled:

```tsx
const [pressed, setPressed] = createSignal(false);

<Toggle pressed={pressed()} onPressedChange={setPressed}>
  <BoldIcon />
</Toggle>
```

Disabled:

```tsx
<Toggle disabled>
  <BoldIcon />
</Toggle>
```

With Indicator:

<InlineCode>ToggleIndicator</InlineCode> renders different content based on the toggle state — the <InlineCode>fallback</InlineCode> is shown when off, and the children are shown when on.

```tsx
<Toggle>
  <ToggleIndicator fallback={<SunIcon />}>
    <MoonIcon />
  </ToggleIndicator>
</Toggle>
```
